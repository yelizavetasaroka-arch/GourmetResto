# GourmetResto — Сайт ресторана

## Описание проекта
Веб-сайт для ресторана "GourmetResto" с меню блюд и информацией о заведении.

## Используемые технологии
- HTML5, CSS3, JavaScript
- Node.js, npm
- Git, GitHub
- Figma

## Структура проекта
GourmetResto/
├── css/ # Стили проекта
│ └── style.css # Основной файл стилей
├── js/ # JavaScript файлы
│ └── script.js # Основной скрипт
├── assets/ # Дополнительные ресурсы
├── images/ # Изображения для сайта
│ └── dishes/ # Фотографии блюд
├── design/ # Макеты и ссылки на Figma
│ ├── figma-link.txt # Ссылка на макет
│ └── images/ # Экспортированные из Figma ресурсы
├── .vscode/ # Настройки VS Code
│ └── settings.json # Настройки форматирования
├── index.html # Главная страница
├── package.json # Зависимости npm
├── .gitignore # Исключения для Git
├── .prettierrc # Настройки Prettier
└── README.md # Описание проекта
